# Credentials Format

## Algorithmia

File: `algorithmia.json`

```
{
  "apiKey": "YOUR API KEY"
}
```

## Watson Natural Language Understanding

File: `watson-nlu.json`

```
{
  "apiKey": "YOUR API KEY",
  "url": "YOUR URL HERE"
}
```

## Google API

File: `google-search.json`

```
{
  "apiKey": "YOUR API KEY HERE",
  "searchEngineId": "YOUR CUSTOM SEARCH ID"
}
```
