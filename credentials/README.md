# Credentials Format

## Algorithmia

File: `algorithmia.json`

```
{
  "apiKey": "YOU API KEY"
}
```

## Watson Natural Language Understanding

File: `watson-nlu.json`

```
{
  "apiKey": "YOU API KEY",
  "url": "YOU URL HERE"
}
```

## Google API

File: `google-search.json`

```
{
  "apiKey": "YOU API KEY HERE",
  "searchEngineId": "YOU CUSTOM SEARCH ID"
}
```
